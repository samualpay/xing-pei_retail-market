# xing-pei_retail-market

## DEMO Website
- [link](https://xing-pei-retail-market.herokuapp.com/)


## Run at local
### Using NPM
1. install mongo
2. setting port 27017
3. create user "root:root" AuthDB:"admin"
4. run command
```
npm install
npm run heroku-postbuild
npm start
```

### Using Docker-Compose
1. check port 27017 not be used
2. run command
```
docker-compose up -d
```
